# Verb Rain

Videojuego web educativo para practicar el **Simple Past** y el **Past Participle** de verbos irregulares en inglés.

## Características

- 3 niveles con 100 verbos cada uno.
- 20 verbos diferentes por sesión.
- Preguntas aleatorias de Simple Past y Past Participle.
- Cuatro palabras que caen en cada ronda.
- Controles con teclado, mouse y pantalla táctil.
- Sistema de puntuación, energía y rescate.
- Lista final de verbos para repasar.
- Récord guardado en el navegador con `localStorage`.
- No requiere librerías externas.

## Abrir en Visual Studio Code

1. Descomprime la carpeta `verb-rain-game`.
2. Abre Visual Studio Code.
3. Selecciona **File > Open Folder** y elige la carpeta.
4. Abre `index.html`.
5. Para probarlo, puedes abrir `index.html` directamente en el navegador.
6. Recomendado: instala la extensión **Live Server**, haz clic derecho en `index.html` y selecciona **Open with Live Server**.

## Controles

- Flecha izquierda o `A`: mover a la izquierda.
- Flecha derecha o `D`: mover a la derecha.
- Mouse o pantalla táctil: mover la canasta.
- Barra espaciadora o `Esc`: pausar o continuar.

## Archivos

- `index.html`: estructura de las pantallas.
- `css/styles.css`: diseño adaptable.
- `js/verbs.js`: base de 300 verbos.
- `js/game.js`: mecánica del videojuego.

## Publicar en GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Sube todos los archivos respetando las carpetas.
3. Abre **Settings > Pages**.
4. En **Build and deployment**, selecciona **Deploy from a branch**.
5. Elige la rama `main` y la carpeta `/root`.
6. Guarda los cambios y espera a que GitHub muestre la dirección pública.
